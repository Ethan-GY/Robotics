# Modern Robot Learning From Scratch

本仓库收录了现代机器人学习的三个核心项目，旨在通过从零实现经典模型，深入理解背后的数学原理与架构逻辑。

## 🚀 项目概览

| 项目名称 | 核心技术 | 应用场景 |
| :--- | :--- | :--- |
| **VAE (变分自编码器)** | 重参数化技巧、隐空间建模 | 图像生成、特征降维 |
| **MDN (混合密度网络)** | 高斯混合模型、不确定性建模 | 机器人决策、多模态回归 |
| **Transformer (Seq2Seq)** | 自注意力机制、位置编码 | 序列翻译、逻辑重组 |

---

## 🛠 项目详情

### 1. VAE: 隐空间学习与图像生成
*   **核心逻辑：** 通过编码器将数据压缩至隐空间，并利用重参数化技巧（Reparameterization）实现概率分布采样。
*   **亮点：** 实现了隐空间的可视化，直观展示了模型如何将无序数据转化为具有语义结构的分布。

### 2. MDN: 解决机器人避障中的“平均值陷阱”
*   **核心逻辑：** 针对传统 MSE 回归在多解问题上的只能取均值的失效（撞墙），使用 MDN 输出概率分布，让模型学会“预测多种可能性”。
*   **亮点：** 验证了为什么在复杂决策任务中，概率预测比点估计（Point Estimate）更具鲁棒性。

### 3. Transformer: 序列到序列的逻辑转换
*   **核心逻辑：** 从零构建 Transformer 架构，实现“颜色+物体”到“物体+颜色”的逻辑对齐。
*   **亮点：** 完整实现了多头自注意力机制、位置编码、Teacher Forcing 训练策略，复现了大模型的核心架构。
